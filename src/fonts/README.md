# Arista-Pro Font Files

Place your Arista-Pro font files here:

## Required Files:
- `AristaPro-Regular.woff2` (Weight: 400)
- `AristaPro-Medium.woff2` (Weight: 500)
- `AristaPro-SemiBold.woff2` (Weight: 600)
- `AristaPro-Bold.woff2` (Weight: 700)

## Where to get Arista-Pro:
- Purchase from: https://www.myfonts.com/fonts/latinotype/arista-pro/
- Or: https://fonts.adobe.com/fonts/arista-pro

## Alternative Formats:
If you have `.ttf` or `.otf` files, you can convert them to `.woff2` using:
- https://cloudconvert.com/woff2-converter
- Or update `src/app/layout.tsx` to use your font format

## After adding fonts:
The font will automatically be applied to "Vaishnavi Organics" in the header.
